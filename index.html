Vue 查漏补缺

  1. data 为什么被定义成函数

    避免组件中的数据互相影响。同一个组件被复用多次会创建多个实例，如果 data 是一个对象的话，这些实例用的是同一个构造函数。为了保证组件的数据独立，要求每个组件都必须通过 data 函数返回一个对象作为组件的状态。

使用递归创建从大到小排列的数组

function countdown(n) {
  if (n < 1) {
    return []
  } else {
    const arr = countdown(n - 1)
    arr.unshift(n)
    return arr
  }
}
